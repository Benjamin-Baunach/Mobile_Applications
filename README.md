# Vue 3 + Vite + Tailwind CSS + Shadcn UI + Javascript


## Start with cloning this repo on your local machine:
```bash
$ git clone https://github.com/Benjamin-Baunach/Mobile_Applications.git
$ cd PROJECT
```

## 💿 Installation
### To install and set up the library, run:
```bash
$ npm install 
$ npm run dev
```
